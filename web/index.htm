nolinks

- A tool that lists all articles with no external links.

The basic premise for all articles on Wikipedia, is that the content must be verifiable and the subject notable. Current policy requires at least two reliable sources, thus requiring external sources. There are lots of templates used to tag articles with missing/incorrect/non-reliable sources, but as far as I know nothing to find articles that has no sources. Hence this tool.

This is not live data, so please note that the list may contain articles which has been fixed. Dumps will be taken around once per week, although the frequency might increase if the resources and time allows. Due to this lag sources might have been added to articles, so please use the "remove from list" button if you've fixed an article, or spot an error.

Please note that I've removed all pages with the words 'disambiguation', 'series' and 'season' in them. Please let me know if there are any other that should be removed, or if you have any feedback (good or bad!) Thanks. - Bjelleklang
