nolinks

- A tool that lists all articles with no external links.

The reasoning behind this tool is that all articles must be verifiable and the subject notable, and thus needs external sources. There are lots of templates used to tag articles with missing/incorrect/non-reliable sources, but as far as I know nothing to find articles that has no sources. Hence this tool.

This is not live data, so please note that the list may contain articles which has been fixed. Dumps will be taken around once per week, although the frequency might increase if the resources and time allows. Please use this, and let me know if you have any feedback, good or bad! Thanks. - Bjelleklang
